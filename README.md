# react-antd-ts

该项目是通过 react+antd+ts 进行开发的项目
该项目包括：

1. react 分支：react+ts 后台管理模板 测试分支
2. react-dev 分支：react+ts 后台管理模板 开发分支
3. react-login-ts 分支：微前端 登陆模块 分支
4. react-photoGallery 分支：微前端 图片库模块 分支

当前分支：react-login-ts
